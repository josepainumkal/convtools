PRMS

Upload your PRMS data here to generate and download a NetCDF translation of your data. Eventually your data and the NetCDF version of your data will both be automatically inserted to the virtual watershed. Once we have that done, we will make it so that you can actually run PRMS on the data you upload.

For more information on PRMS see the USGS's PRMS documentation.

  • input file holds the PRMS model parameters. The values in the file represent the values of parameters for hydrologic, climate, or other modules used in PRMS.
  • HRU (Hydrologic Response Units) is the model spatial discretization. The values in hru file represents the location of stream cells in PRMS.
  • Number of rows implies the number of rows of the dataset.
  • Number of columns implies the number of columns of the dataset.
  • EPSG value implies the EPSG code of the dataset used for the projection. By default, the value is set as 4326.
